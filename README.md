## Кваліфікаційна робота освітнього ступеня бакалавр
ВНЗ: Черкаський національний університет імені Богдана Хмельницького

Факультет: Факультет обчислювальної техніки, інтелектуальних та управляючих систем (ФОТІС)

Кафедра: Кафедра інформаційних технологій

Спеціальність: 122 – Комп'ютерні науки (КН)

Тема роботи: **"ІНФОРМАЦІЙНА СИСТЕМА БУДІВЕЛЬНОЇ КОМПАНІЇ ДЛЯ ПОКУПЦІВ"**

Здобувач: Хало Ігор Сергійович (група КН-22)

Керівник: кандидат технічних наук, старший викладач Царик Т.Ю.

Рік захисту: 2026

### Технологічний стек
- C# / .NET 8
- ASP.NET Core Razor Pages
- Entity Framework Core (Code-First)
- SQL LocalDB
- Bootstrap 5
- ASP.NET Core Identity + BCrypt

### Вміст репозиторію
- Файли проекту ConstructionCompany
- Пояснювальна записка кваліфікаційної работи у форматах Microsoft Word (DOCX) та PDF
- Презентація у форматі Microsoft PowerPoint (PPT)
- Файли для створення інсталятора за допомогою Inno Setup 6 (факультативно)

### Структура каталогів
- ConstructionCompany (містить файли проекту для компіляцію за допомогою MS Visual Studio)
- docs (містить файли пояснювальної записки та презентацію)
- installer (містить файли для Inno Setup 6)
